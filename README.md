# Renombrar-Carpetas

cambia el nombre de carpetas existente de forma automatica con python.

Pasos:
1. Import os.
2. Obtener la ruta de carpeta donde se encuentran los archivos.
3. Se agrega prefijo en el nombre de cada archivo.
4. Se obtiene la ruta completa del archivo.
5. Se Valida si es o no una carpeta dentro de la carpeta principal.
6. Se obtiene el nuevo nombre.
7. Se obtine la ruta completa del nuevo archivo.
8. Y finalmente se renombra el archivo.
